# 个人主页 (Personal Portfolio)

这是一个简洁、专业且响应式的个人主页模板，专为求职和个人展示设计。

## 功能特点

- **响应式设计**：完美适配桌面、平板和移动设备。
- **简洁风格**：专注于内容展示，视觉体验舒适。
- **主要板块**：
    - 英雄区 (Hero Section)：自我介绍与行动号召。
    - 关于我 (About)：个人简介与核心数据。
    - 技能 (Skills)：技术栈展示。
    - 经历 (Experience)：工作与教育时间轴。
    - 项目 (Projects)：精选项目展示卡片。
    - 联系 (Contact)：社交链接与联系方式。
- **交互效果**：平滑滚动、移动端汉堡菜单、滚动渐显动画。

## 快速开始

1.  **克隆或下载**此仓库。
2.  直接在浏览器中打开 `index.html` 文件即可预览。
3.  **部署**：
    - 将代码推送到 GitHub 仓库。
    - 在仓库设置 (Settings) -> Pages 中选择 `main` 分支作为源。
    - 保存后，你的个人主页将在 `https://yourusername.github.io` 上线。

## 如何自定义

所有内容都旨在易于修改。

### 1. 修改个人信息
打开 `index.html`，搜索并替换以下内容：
- `姓名/昵称`：替换为你的名字。
- `软件工程师 / 全栈开发者`：替换为你的职位。
- `your.email@example.com`：替换为你的真实邮箱。
- 社交链接 (`href="..."`)：替换为你的 GitHub、LinkedIn 等主页链接。

### 2. 更新图片
- 在 `projects` 区域，你可以用真实的 `<img>` 标签替换 `<div class="img-placeholder">`，展示你的项目截图。
- 建议图片尺寸：600x400px 或 16:9 比例。

### 3. 修改配色
打开 `style.css`，在文件顶部的 `:root` 区域修改颜色变量：

```css
:root {
    --primary-color: #2563eb; /* 主色调 (蓝色) */
    --secondary-color: #1e40af; /* 悬停深色 */
    /* ... */
}
```
