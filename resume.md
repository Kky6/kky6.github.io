## 项目经历

### 1. 大模型驱动的地质成矿知识图谱构建
- **时间**：2025.09 - 2026.02
- **项目介绍**：
  针对传统地质知识抽取模型依赖预定义标签集且高质量标注数据稀缺的问题，利用大语言模型的 Zero-shot/Few-shot 能力，设计并实现了一套从非结构化文献到结构化成矿知识图谱的自动化构建系统。
- **核心工作**：
  - **两阶段 (Two-Stage) 知识抽取架构设计**：将命名实体识别（NER）解耦为“边界抽取”与“上下文分类”两个独立阶段，降低推理幻觉与漏抽取率。
  - **大模型后训练 (Post-Training) 与对齐**：基于合成样本进行指令微调 (SFT) 和基于人类反馈的强化学习 (RLHF)，分别训练专属的 Annotator 和 Classifier。
  - **In-Context Learning (ICL) 优化**：设计 Few-shot 演示检索模板，激发模型在特定地质场景下的上下文学习能力。
- **成果**：
  - 两阶段法相较于单阶段法 F1 值提升近 10%（Qwen3-8B 提升 9.94%）。
  - 优化后的 Qwen3-32B 模型在 17 个地质实体类别分类任务中，F1 分数超越千亿参数基座模型（Qwen3-235B 为 77.14%）。

### 2. 百万级地质文献智能筛分 Agent 工作流系统
- **时间**：2025.09 - 2025.10
- **项目介绍**：
  作为“成矿知识图谱”项目的前置数据引擎，旨在从海量非结构化历史地质 PDF 文献中，全自动挖掘、清洗并评级出高价值的成矿核心语料。
- **核心工作**：
  - **多 Agent 工作流编排**：基于 LangGraph 框架设计 Agentic Pipeline，解耦为“多模态特征提取 -> 大类初筛 -> 细分领域判别 -> 核心价值评级”四个节点。
  - **异构大模型路由与协同**：动态调度不同规模模型。使用 Qwen-VL-8B 提取视觉信息，Qwen-32B 粗分类，Qwen-30B-Thinking 深度研判。
  - **高并发部署与推理优化**：在 8 张 A100 (80G) GPU 集群上采用 vLLM 框架分布式部署，48 小时内完成 135.5 万篇文献洗筛，吞吐量 0.13 秒/篇。
  - **专家级提示词工程**：设计包含复杂逻辑判断的分级 Prompt 体系。
  - **数据清洗与解析**：引入 MinerU (VLM) 解析 PDF，构建高质量结构化 JSON 语料库。
- **成果**：
  - 从 135 万篇文献中精准提纯出 7.6 万篇成矿相关文献。
  - 严格划分为 A（核心，7.5%）、B（辅助，12.4%）、C（低相关，80.1%）三个层级，自动生成结构化 CSV 与分析报告。
